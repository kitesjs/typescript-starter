This is basic project starter repository.

```bash
# Clone project template using git
$ git clone https://github.com/kitesjs/typescript-starter.git
```

# Installation

```bash
$ npm install
```

# Development

```bash
# Start project in development
$ npm start

# Start project in production
$ npm run start:prod
```
